profile Page

Technical Description

The Sitemap Page provides a **visual representation** of the website structure, allowing users to easily navigate between different sections. It includes an **interactive SVG diagram** that links to various pages, making it a useful tool for both users and developers to understand the website’s layout.

Structure & Layout: The page follows the same **consistent structure** as other pages on the website. It includes a <header> with a logo and navigation bar, a **main section** displaying the sitemap diagram, and a <footer> with copyright information and a link to the Page Editor. The **SVG-based diagram** visually maps out the website, with each section connected by lines for clarity.

Key Styling Choices: - The **SVG sitemap** provides an easy-to-follow **flowchart-style layout**, with rectangular nodes representing each page. - **Lines and nodes are styled** to create a clean, visually appealing map of the website. - **Hover effects and clickable links** allow users to jump directly to specific pages by clicking on the relevant sections. - The **navigation menu and footer** match the global website design for consistency.

Accessibility

- **ARIA labels** and title attributes are included in the SVG elements to ensure screen readers can describe each page link. - **Clear text labels** inside each node help users quickly identify different sections of the website. - The **SVG format** ensures scalability, making it readable on different screen sizes and resolutions.

sitemap Page

Technical Description

The Sitemap Page is an essential part of the website, providing a **visual overview** of the site's structure through an **interactive SVG diagram**. It helps users and developers understand the relationship between different pages and how they connect. The **interactive elements** allow users to navigate directly to key sections of the site with a single click.

Structure & Layout: The page follows a **consistent design** with the rest of the website, including a <header> with a logo and navigation menu, a **main section** containing the sitemap diagram, and a <footer> with copyright details and a link to the Page Editor. The **SVG-based diagram** presents the website hierarchy, with clear lines connecting the different pages, making navigation intuitive.

Key Styling Choices: - The **SVG sitemap** follows a **structured flowchart-style layout**, improving clarity. - **Nodes (rectangles) represent different pages**, ensuring each section is visually distinct. - **Lines and text are styled** for readability and aesthetic appeal. - The **navigation menu and footer** maintain uniformity with other pages, reinforcing website consistency.

Accessibility

- **ARIA labels and attributes** are used to ensure screen readers can describe the sitemap’s contents. - **Clear, readable text labels** inside each node enhance usability for all users. - **SVG scalability** allows the diagram to adapt to different screen sizes without losing clarity.

Content Page

Technical Description

The Content Page is designed to provide detailed information about **coral reefs** and their importance. It follows a structured layout with an **interactive sidebar**, **informative sections**, and **external resources**, ensuring users can easily navigate and explore relevant content.

Structure & Layout: The page follows the website’s **global design**, including a <header> with a logo and navigation menu, a **main content section**, and a <footer> for branding. The **sidebar navigation** allows users to jump to different sections instantly, improving accessibility. The main content is divided into **two key sections**: - **Why Coral Reefs Matter:** Provides a clear explanation of their ecological and economic significance. - **Resources:** Offers links to trusted organizations and research materials on coral reef conservation.

Key Styling Choices: - The **sidebar navigation** remains fixed, allowing users to navigate without scrolling back to the top. - **Cards** are used for content sections, giving the page a **structured and visually appealing** look. - The **main heading** is styled to stand out and establish a clear topic focus. - An **image** is included within the content to make the page more engaging and informative.

Accessibility

- **Alt text for images:** Ensures visually impaired users can understand the visuals. - **Clear navigation links:** Allows users to quickly access important sections of the page. - **External resource links:** Provide additional learning opportunities from reliable sources.
